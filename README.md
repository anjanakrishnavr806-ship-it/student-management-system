# Student Management System

A simple **Django-based web application** to manage student information efficiently.  
You can add, update, delete, and view student records.

## Features

- Add new students
- Update student details
- Delete student records
- View all students

## Tech Stack

- Python, Django  
- HTML, CSS  
- SQLite (default database)
